# One Nighters Army

> For students who open the syllabus when the exam opens fire.

**One Nighters Army** is a web-first study survival prototype for last-night exam warriors. It turns a subject syllabus into a battle plan: unit-wise checklists, formula vaults, weak-topic warnings, progress dashboards, and survival-mode revision flows.

## Why this exists

Most students do not need another boring to-do list. They need a panic-proof command center that says:

- What should I study first?
- What can I skip if time is dying?
- How much progress have I actually made?
- Which formulas should I revise before entering the exam hall?

This web MVP is the first proof-of-concept before the Android app.

## MVP Features

- Premium landing page
- Fake syllabus upload demo
- Auto-generated battle plan preview
- Unit-wise interactive checklist
- Survival dashboard with progress stats
- Formula vault preview
- Local-only demo data

## Tech Stack

- React
- Vite
- Framer Motion
- Lucide React
- Plain CSS with glassmorphism design system

## Local Setup

```bash
npm install
npm run dev
```
